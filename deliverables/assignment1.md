Description du sujet

Le projet consiste à construire un modèle de machine learning supervisé de classification permettant de prédire si un client présente un comportement de fraude énergétique ou non. Le modèle s’appuie sur des variables décrivant le comportement du client, comme les anomalies de facturation, les relevés de compteur ou les retards de paiement, afin d’identifier les profils suspects.

Première idée de problématique

Comment exploiter les données comportementales des clients pour détecter efficacement les cas de fraude énergétique, tout en minimisant les erreurs, en particulier les faux négatifs qui correspondent à des fraudes non détectées et représentent une perte financière pour l’entreprise ?

Dataset idéal

Le dataset utilisé contient plusieurs variables décrivant le comportement des clients, notamment des indicateurs d’anomalies de facturation, de relevés de compteur, de retards de paiement et de caractéristiques régionales. La variable cible permet de distinguer les clients normaux, les cas de fraude potentielle et les fraudes confirmées. Pour un problème de classification binaire, ces classes peuvent être regroupées en deux catégories : fraude et non fraude.
